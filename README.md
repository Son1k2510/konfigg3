# Ассемблер для Учебной Виртуальной Машины (УВМ)

## Этап 1: CLI ассемблер
- Принимает аргументы: исходный CSV, бинарный файл, флаг --test
- CSV-формат языка ассемблера
- Поддерживает команды: LOAD_CONST, READ_MEM, WRITE_MEM, SGN
- В режиме тестирования выводит промежуточное представление

## Этап 2: Формирование машинного кода
- Трансляция промежуточного представления в машинный код
- Запись результата в двоичный файл
- Вывод размера файла в байтах
- В режиме тестирования: вывод байтового представления
- Тестовые последовательности соответствуют спецификации УВМ

### Пример использования:
```bash
python assembler.py program.csv program.bin --test
```
